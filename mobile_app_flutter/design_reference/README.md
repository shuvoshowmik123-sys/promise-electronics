# Design Reference

This folder contains reference images and code snippets for the TV ডাক্তার Flutter app UI design.

## Folder Structure

```
design_reference/
├── images/           # Reference images and mockups
├── code_snippets/    # Reusable UI code examples
└── README.md         # This file
```

## Design System

- **Theme**: Light Mode
- **Primary Color**: `#36e27b` (Fresh Green)
- **Background**: `#FFFFFF` (Pure White)
- **Typography**: Outfit (headings), Inter (body)

## Screens to Design

1. Home Screen (Hero + Bento Grid)
2. Daktar Vai (AI Chat)
3. Repair Request (Multi-step form)
4. Shop (Product grid)
5. Repair History (Card list)
6. Profile (Settings menu)

## Usage

Drop your reference images into the `images/` folder and code snippets into `code_snippets/`.
